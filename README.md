📊 Student Performance Analysis Dashboard (Power BI)
📌 Project Overview

The Student Performance Analysis Dashboard is a data analytics project built using Power BI to analyze and visualize student academic performance. The dashboard helps identify key insights such as pass/fail rates, subject-wise performance, and overall academic trends.

This project demonstrates how data visualization and business intelligence tools can be used in the education sector to support data-driven decision making.

🎯 Objectives

Analyze student academic performance using interactive dashboards

Identify pass and fail rates across subjects

Track student performance trends

Provide visual insights for educators and administrators

Demonstrate Power BI data modeling and visualization skills

📂 Dataset

The dataset contains information related to student academic records including:

Student ID

Student Name

Subject Scores

Total Marks

Result (Pass / Fail)

Performance indicators

The data was cleaned and processed before visualization.

⚙️ Tools & Technologies

Power BI – Data visualization and dashboard creation

Microsoft Excel – Dataset storage and preprocessing

Data Modeling – Relationships and calculated measures

DAX (Data Analysis Expressions) – Custom metrics and KPIs

📊 Dashboard Features

The Power BI dashboard includes the following insights:

📈 Overall Student Performance Overview

✅ Pass vs Fail Analysis

📚 Subject-wise Performance

📊 Average Marks Analysis

🎛 Interactive Filters (Slicers)

🧑‍🎓 Student-wise Performance Tracking

Users can interact with filters to explore performance across different dimensions.

🔑 Key Metrics (KPIs)

Total Students

Pass Count

Fail Count

Pass Percentage

Average Marks

These KPIs help quickly understand overall academic performance.
